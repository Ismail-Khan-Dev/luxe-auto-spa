**Luxe Auto Spa** is a high-end, conversion-focused web application designed for elite automotive detailing services. Built with a "Mobile-First" philosophy and premium aesthetics, it delivers a luxury brand experience from the first pixel.

---

## ✨ Key Features

- 📱 **Fully Responsive** — Seamless experience from mobile to ultra-wide displays.
- ⚡ **Turbo Performance** — Optimized with Vite for near-instant load times.
- 🎨 **Luxury UI/UX** — Glassmorphism effects, premium typography, and smooth GSAP/Framer Motion animations.
- 🗓️ **Service Showcases** — Detailed car care packages with clear CTAs.
- 📍 **SEO Optimized** — Built-in semantic HTML and metadata for local search dominance.

## 🛠 Tech Stack

- **Core Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 7](https://vite.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [Radix UI](https://www.radix-ui.com/) (Primitive-based accessibility)
- **Animations**: [GSAP](https://gsap.com/) & [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18.x or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Ismail-Khan-Dev/luxe-auto-spa.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

---

## 🏗 Production Build

To create an optimized production build:

```bash
npm run build
```

The output will be generated in the `dist/` directory, ready to be served.

---

## 🌩 Vercel Deployment

This project is optimized for **Vercel** with zero-config deployment:

1. Push your changes to the `main` branch.
2. Connect your GitHub repository to Vercel.
3. Use the following build settings (Detected automatically):
   - **Framework Preset**: Vite
   - **Build Command**: `tsc -b && vite build`
   - **Output Directory**: `dist`
